
# Age & gender Classification





## Run Locally

Clone the project

```bash
  git clone https://github.com/vansh7071/Age-gender-Classification.git
```

Go to the project directory

```bash
  cd Age-gender-Classification
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Launch project

```bash
  python -u launch.py
```
